This folder contains display drivers:

qvga ... output to QVGA monitor (VGA timmings) with resolution
	320x240 pixels, 8 bits per pixel in RGB332 format

st7789 ... ST7789 controller with output to TFT display with resolution
	320x240 pixels, 16 bits per pixel in RGB565 format

vga ... output to VGA monitor (VGA timmings) with resolution
	320x240 pixels, 16 bits per pixel in RGB565 format
