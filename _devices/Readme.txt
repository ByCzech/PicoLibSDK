This folder contains definitions of supported devices:

demovga ... DemoVGA board, with output to VGA display 320x240/16-bit
pico ...... simple Raspberry Pico module
picoino ... Picoino computer, with output to QVGA display 320x240/8-bit and 56-key keyboard.
picopad ... PicoPad console, with output to TFT display 320x240/16-bit
